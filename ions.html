Background

The above example shows a coordination visualization in D3 using the live properties Architecture. The data for the following visualization is gathered from Ions Example. For the above visualization we created a wrapper for the D3 code and modified the code to support the visualization.

Ions Visualization Interface

The visualization displays different spatial and temporal aspects of a simulated ion cloud trajectory. A highly coordinated set of timeseries plots, table views and other views reveal different characteristics of oscillatory motion. The trajectory is displayed in a variety of ways by the views in the visualization:

  • Three time series scatterplots show ion motion over time relative to each pair of trap plates (excite, detect, inject).
  • Monoscopic stereo views show the trajectory and selected subtrajectory next to the corresponding scatterplot matrices.
  • The parallel coordinate plots show the same trajectory.
  • A table view shows the actual {T,X,Y,Z} positions of points along the trajectory.
  • The Perpectiual view below the Table view shows the Time mapped to the linear gradient of the view.

All views map time into a translucent red-to-black color gradient that suggests (but does not map precisely to) decreasing energy of the ion cloud over time.
