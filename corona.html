Corona - Example

Background

The above example shows a coordination visualization in D3 using the live properties Architecture. The data for the following visualization is gathered from Corona Example. For the above visualization we created a wrapper for the D3 code and modified the code to support the visualization.

Corona Visualization Interface

The visualization shows two scatterplots, table, Choropleth map, bubble chart and a parallel coordinate plot coordinated in multiple ways.

  • The Top Left scatterplot shows Tests Per Million drawn on horizontal axis vs Cases Per Million on vertical Axis
  • The Bottom Left scatterplot shows Deaths Per Million drawn on horizontal axis vs Tests Per Million on vertical Axis
  • The Table view shows all the data set. THe Continent coloumn is colored based on the categorical color legend assigned to continent.
  • The Choropleth Map shows a Mercator projection of the world map. Each continent is colored based on the categorical color legend assigned to continent
  • The Bubble chart view shows various bubbles drawn using size to represent the size of the circle
  • The Parallel coordinate is drawn using the same data. The color of the paths is drawn using categorical color legend assigned to attribute of the continent

Coordination and Interaction

The two scatterplots are coordinated by the reange property. The Horizontal range of the top scatterplot is coordinated with the vertical range of bottom scatterplot. The Top Left scatterplot and the Map are coordinated in selection. Any selection made in either of the above will reflect in other. The Bottom Left scatterplot, Table view and the Parallerl coordinate plot are coordinated in selection. Any selection made in either of the above will reflect in other. The scatterplots, table view, Choropleth map and bubble chart are coordinated in indication. Any item hovered in table view, Choropleth map and bubble chart will be highlighted in other plots.

All the views use the color encoding from color brewer website to encode the color with the continent that the attribute belongs to. Selection is shown in scatterplot using a black colored stroke ove the circles. Indication is shown in the scatterplot using a black colored circle with a label. Selection in Table view is done using the red colored font for the row while indication is shown using teal colored background for the view. Selection in choropleth map is done using the dark grey colored region for the row while indication is shown using blue colored outline for the region. Selection in Bubble chart is done using the grey colored circles for the bubble while indication is shown using black colored stroke for the for the bubble. Selection in Parallel coordinate plot is done by increasing the opacity of the paths to 100% while reducing the opacity of the unselected items to 50%.
